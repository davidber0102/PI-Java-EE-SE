----------------------modelo vista controlado (MVC)----------------------------------------------------

¿QUE ES MVC ?
	PATRON DE DISELO A ALA HOTA DE CREAR APLICACIONES DE SERVIDOR JAVA

¿EN QUE CONSISTE ESTE PATRON DE DISEÑO?
	SEPARAR LA PARTE LOGICA DE LA PARTE VISUAL EN NUESTROS PROYECTOS
	¿COMO LO HACE EL PATRON MVC?
		INTEGRANDO sERVLETS Y jsp EN NUESTROS PROYECTOS HACIENDO QUE TRABAJEN JUNTOS
			LOS SERVLETS SE ENCARHGAR DE LA PARTE LOGICA
			JSP SE ENCARGARA DE LA PARTE VISUAL

MODELO VISTA CONTROLADO


	1 PETICION DE USUARIO ( LLENADO DE FORMUALRIO)	
	2 MEDIANTE EL CONTROLADOR SE HACE PETICION A BBDD
	3 EL CONTROLADO MANDA INFORMACION AL ARCHIVO VISTA ------- INTERFACE REQUESTDISPATCHER------ METODO FORWARD()
	4 ARCHIVO DE RESPUESTA
	
	MODELO ------------- BBDD
	CONTROLADOR ---------SERVLET
	VISTA --------------- JSP

--------------------VENTAJAS MVC
	ELIMNAR CODIGO HTML DEL SERVLET
	ELIMINAR CODIGO JAVA EN JSP ( NO MAS SCRIPLETS)
	FAVORE EL DESAROOLLO POR VARIOS PROGRAMADORES ---- MODULARIZACION
	FAVORE LA DEPURACION DE ERRORES
	RAVORES LA ESCALABILIDAD DEL PROYECTO


----------------CRUD-------------------

	C ---- CREATE INSERT
	R ---- READ - LEER
	U ---- IPDATE ACTYALIZAR
	D ----- DELETE - BORRAAR


POOL DE CONEXIONES--- 
- PASOS A SEGUIR
	1.- DESCARGAR EK DRIVER JDBC PARA MYSQL (YA LO VIMOS CON LOS VIDEOS DE JDBC)
	2.- COLOCALR EL DRIVER EN "WEB-INF/LIB"
	3.- dEFINIR (CONSTRUIR) EL POOL DE CONEXIONES EN META-INF/contex.xml"
	4.- CREAR UNA REFERENCIA AL POOL DE CONEXION DESDE CODIGO JAVA
